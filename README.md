## How to Deploy

Want to create your own gaming fan site? Here's how:

1. **Fork the Repository**

```bash
git clone https://github.com/yourusername/chopchains.com.git
cd chopchains.com
```

2. **Run Locally**

```bash
npx serve
```

The site will be available at `http://localhost:3000`

3. **Deploy to Static Host**

Recommended hosting options:

- **Vercel**: `vercel --prod` (recommended for automatic deployments)
- **Cloudflare Pages**: Connect GitHub repo for automatic deployments
- **GitHub Pages**: Enable Pages in repository settings
- **Netlify**: Connect repo for continuous deployment
